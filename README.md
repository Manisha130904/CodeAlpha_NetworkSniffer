# CodeAlpha_NetworkSniffer

A Python-based network packet sniffer built using **Scapy** that captures and analyzes live network traffic (TCP/UDP) on a system in real-time.

## 🔍 Features
- Captures live network packets
- Identifies source & destination IP addresses
- Detects protocol type (TCP / UDP / Other)
- Displays source and destination ports for TCP/UDP traffic

## 🛠️ Tools & Technologies
- Python 3.14
- Scapy library
- Npcap (for packet capture on Windows)

## ▶️ How to Run
1. Install dependencies:
    pip install scapy
2. Run the script **as Administrator**:
python sniffer.py
3. Press `Ctrl+C` to stop the sniffer.
